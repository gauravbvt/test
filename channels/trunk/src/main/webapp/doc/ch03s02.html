Logging in and accessing a Plan and segment

When accessing Channels, an unrecognized user is presented with a login screen. The user is requested to enter user name and password. To avoid being asked to login again within the current browser session, the user can check the "Remember me..." check box (the browser must allow cookies for this to work). Upon successful login, a user with Planner privileges will be taken to the Plan editor. A user without Planner privileges is taken to his/her Procedures.
