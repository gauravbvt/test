Collaboration tab panels

The collaboration tab panels on the right side of the Plan Editor allow a planner see who else is working on the plan and what changes they made recently. A planner can send messages to a single planner or to all users and can read received messages. The collaboration tab panels can be collapsed if more screen space for the flow map and task/flow forms is needed.

Presence tab

The presence tab by default shows users currently logged in and who selected the current plan (and version) from their Home pages. The presence tab is automatically and regularly refreshed. A planner can ask to see all registered users whether or not they are logged in. A message can be addressed to a planner from the "more" menu.
