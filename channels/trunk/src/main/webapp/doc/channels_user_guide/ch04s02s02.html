Completeness issues

An element of the IS plan is clearly missing given the scope of the plan and the elements it contains.

Cause

Severity

Remediation

Can be waived

Agent not member of any known organization

Low

  • Make the agent the member of an organization.

No

Sharing agreement encompassed by another

Low

  • Remove either sharing agreement.

Yes

Sharing agreement does not match any sharing commitment

Low

  • Remove the sharing agreement or add flows that imply a commitment covered by the agreement.

Yes

Immediate or permanent task is unnecessarily triggered

Low

  • Have no flow trigger this task or have the task not start with the plan segment.

Yes

Organization expected to have task assignments but has none

Low

  • Remove the expectation on the organization or add a task that will be assigned to the organization or one of its sub-organizations, if any.

No

Organization is missing expected commitment

Low

  • Fulfill expectation by having a task of a given category and assigned to an organization connected to another task of an expected category and assigned to an organization with the appropriate relationship, with a flow with an expected intent. For example, an audit task expects to receive a report from an operations task assigned to a child organization.

Yes

Organization is not assigned a category of tasks

Low

  • Specify at least one task of the missing category that is assigned to the organization or one of its sub-organizations.

Yes

Information capability or need has no element of information

Low

  • Add elements of information.

Yes

Sharing flow has no elements of information

Medium

  • Specify elements of information

Yes

Sharing commitment by an organization without required agreement

Low

  • Confirm an agreement covering this sharing commitment or remove the requirement for agreements for the organization.

Yes

Sharing commitment over unicast channel with unknown endpoint

Depends on impacted tasks

  • Make sure that the agent has a known endpoint (address, phone number etc.) for contact via the unicast channel, or remove the unicast channel from the flow.

No

Agent shares information with self

Low

  • Change the agent sharing the information, change the agent receiving the information, or generalize task specifications to assign multiple agents to either the source or target task.

Yes

Information flow specified with a channel

Depends on impacted tasks if sharing, else low

  • Add a channel to the specification.

No

Task never starts

Depends on impacted tasks

  • Start this task with the plan segment or have an incoming sharing flow trigger it.

No

Non-planner user is not a participant

Low

  • Assign an agent to the user or unregister the user.

Yes

No plan segment for event phase

Low

  • Add a plan segment for phase of event, remove event from plan scope, or remove phase from plan scope.

Yes

No plan segment for incident

High

  • Define a plan segment that responds to it or remove the event's incident status.

No

Task neither consumes nor produces information

Depends on task’s importance

  • Add information produced or add received consumed.

No

No known agent in task's named role

Medium

  • Name an agent assigned to the task or add a participating agent to the plan's scope with this role.

Yes

Redundant sharing flow

Low

  • Change the name of information transmitted or break up the sharing flow.

Yes

Redundant task

Low

  • Remove redundant task or specify one of the tasks differently.

Yes

Redundant place

Low

  • Remove references to this place, change its address, or change its geolocation.

Yes

Planned-for event may never be caused

Medium

  • Make the event an incident or make sure at least one task in any segment causes it.

No

Segment’s event phase may never end

Medium

  • If the segment is for a co-event phase, have the event end on its own.

  • If the plan segment is for a pre-event phase, have the event be an incident or have a task in any segment cause it.

  • In all cases, have at least one task in the segment terminate the event phase.

Yes

Segment's event phase may never start

Medium

  • If the segment is for a co-event phase, have the event be an incident, or have the task in another segment cause it.

  • If the plan segement is for a post-event phase, have the event be self-terminating or have a task in a co-event segment terminate its segment.

Yes

Sharing flow implies no commitments

Medium

  • Modify the specifications for source and target tasks so that they both imply assignments or add or redefine participating agents to match source and target specifications.

  • If inadequate clearances are at fault, declassify elements of information or make sure otherwise-committed agents have the required clearances.

Yes

Task is triggered but never started

Depends on impacted tasks

  • If the task starts with its segment (it is permanent or immediate), make sure the event phase of the segment can start (it is a co-event phase of an incident or the phase is started by a task that itself can start), or else make it so the triggering task can start itself.

No

Unconfirmed job of agent in organization

Low

  • Confirm the job in the organization’s profile.

No

Ignored information need or capability

Low to medium

  • Connect need or capability via a sharing flow to/from another task.

No

Unassigned task

Depends on impacted tasks

  • Explicitly assign an agent to the task, profile an agent to match the task specifications, or modify the task specifications so that it matches at least one participating agent.

Yes

Information need not fully satisfied

Depends on impacted tasks

  • Add sharing flows or extend current sharing flows to include the missing elements of information.

Yes

Unsupervised job

Low

  • Add a supervisor agent to the job or remove the job for the employing organization's profile.

Yes

Postal code can't be verified

Low

  • Change the postal code or change the geolocation.

No

Agent is not assigned any task

Low

  • Explicitly assign agent to a task, specify a task that matches the agent's profile, or modify the agent's profile so that it matches one or more task specifications.

Yes

Useless task

Low

  • Have the task achieve a goal, end the event phase (if it would end a risk), or make sure at least one task that it directly or indirectly triggers or critically informs achieves a goal.

Yes

Agent without contact info

High

  • Add a channel via which to contact the agent.

Yes

Confirmed sharing agreement without MOU

Medium

  • Attach an MOU to the sharing agreement or unconfirm the sharing agreement.

No

Redundancy in segment context

Low

  • Remove redundant pre- or post-event from the context.

No

Segment context may never come to be

Medium

  • Make event in context an incident, have a task cause the event, make event self-terminating (post-event only), or make a task terminate the event (post-event only).

No

Flow not standard compliant

Low

  • Add expected element of information or remove information standard tag.

No

No user participates as placeholder agent

Medium

  • Assign a user to the agent or make the agent not a placeholder.

Yes

Required channel address is empty

Depends on impacted tasks or medium

  • Enter a valid address.

No

Agent commits to share with supervisor but has none

Depends on impacted tasks

  • Identify the agent’s supervisor or remove the restriction to share only with a supervisor.

Yes
