Issues Summary Page

Channels summarizes all outstanding issues detected in and manually added to a plan in a page accessible from the Home page. The issues summary page provides metrics about the severities of the three overall types of issues (robustness, completeness, and validity), and about each kind of issue found in the plan. (Channels detects more than 90 different kinds of issues.)
