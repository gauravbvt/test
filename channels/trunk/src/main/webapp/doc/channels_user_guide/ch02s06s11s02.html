Organization profile panel

A planner profiling an actual organization can define its mission and give its location, contact information, and parent organization.

A planner can declare that it would be an issue should the profiled organizations have a sharing commitment with another that is not covered by an agreement. A planner can also declare that it would also an issue should there be a role assigned by the organization in the plan for which there is no known agent.

Figure 2.94. Organization profile panel - details view


A planner can add agents to an organization's membership/employment roster, by naming the agent (required), the agent's role (required), jurisdiction (optional), title (optional) and supervisor agent (optional). Alternately, the planner can confirm memberships that are already implied by task specifications (a task definition can identify an agent by name in a role within a jurisdiction for an organization).

Figure 2.95. Organization profile panel - details view - jobs


Jobs can also be added to an organization by transferring them, or copying them, from another organization.

Figure 2.96. Organization profile panel - details view - transferring jobs


Organizations with parent organizations form hierarchical structures that can be visualized as org charts.

Figure 2.97. Organization profile panel - details view - org chart


Inter-organizational sharing commitments are commitments between agents acting on behalf of different organizations. A sharing agreement may or may not exist for a sharing commitment that crosses organizational boundaries. It is an issue if an inter-organizational sharing commitment is unconfirmed when it is required by the committing organization. When viewing an organization, a planner can see the sharing commitments the organization makes with other organizations and confirm whether each commitment is covered or not by an agreement. A planner can attached documents of type MOU (Memorandum of Understanding) to confirmed sharing agreements. A sharing agreement not documented by a Memorandum of Understanding is an issue.

Figure 2.98. Organization profile panel - details view - sharing agreements


The task assignments and sharing commitments of the organization can be expanded at the bottom of the view. This is followed by an index of plan elements referencing the organization, and by issues about the organization proper, if any.

Figure 2.99. Organization profile panel - details view - expanded analytical content
