Phase profile panel

Planners can add any number of phases to the scope of a plan. In addition to setting generic entity attributes (name, description, classification), a planner indicates whether a phase is before an event (pre-event), during an event (co-event) or after an event (post-event).

Figure 2.119. Phase profile panel - details view


The phase profile panel also has an Issues view to show all issues that relate to a profiled phase.
