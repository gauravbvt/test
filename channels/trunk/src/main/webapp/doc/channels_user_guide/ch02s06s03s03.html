Messages tab

The messages tab enables planners (and only planners) to send personal messages to another user with access to the currently selected plan, or to broadcast messages to all planners or participants.

A planner can show or hide broadcasts (nonpersonal messages), and alternately look at messages received or sent.

Figure 2.42. Messages tab


Messages can be sent "internally," to be read only from within the messages tab, or they can be sent via email. Internal messages can be "resent" by email. Messages sent internally can also be deleted.

A message sent from the context of a plan element will include a reference to that element. The recipient of the message can click on that reference to be taken to the plan element, if the element still exists in the plan.

Figure 2.43. Sending and receiving a message about a plan element (here a task)


Note that the messaging capabilities of Channels should be used only while planning and should not be used during emergencies.
