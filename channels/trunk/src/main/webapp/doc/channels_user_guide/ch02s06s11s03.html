Agent profile

The profile attributes specific to an actual agent are:

  • whether or not the agent is an archetype or a placeholder or neither,

  • whether the agent is a system or a person,

  • the agent's contact information (transmission media with addresses/endpoints where applicable),

  • the agent's availability,

  • and the agent's security clearances (as secrecy classification levels)

In addition to a profile attributes edit form, the Details view of an agent lists the roles played by the agent, the agent's task assignments and sharing commitments (if expanded), all references to the agent (indexed) and all issues detected or reported specifically about the agent.
