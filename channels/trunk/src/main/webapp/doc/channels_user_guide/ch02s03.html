Logging in

When accessing Channels, an unrecognized user is presented with a login screen. The user is asked to enter user name and password. To avoid being asked to login again after quitting without signing out, the user can check the "Remember me..." check box (the browser must allow cookies for this to work).

Upon successful login, a user is taken to the Channels Home Page.

Note that users can not self-register into a Channels server. They must be added together with their initial passwords by a Channels administrator.

A user who has forgotten his/her password or user id can request a new password. The new password together with a user id reminder is sent via email (the user must remember the email address used when registering him/her into the Channels server).
