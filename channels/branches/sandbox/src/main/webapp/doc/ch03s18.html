Collaboration

The collaboration panel allows a planner to see who else is working on the current plan and find out what has done most recently to the Plan (activities). It also allows a planner to send messages to one or all planners, and view messages received. Each message, if sent in the context of a Plan element (task, flow, entity, segment etc.), contains a link to that element. Messaging is done within Channels but a planner may elect to send a copy of a message by email.
