Issue detection and management

Channels automatically detects more than 70 different kinds of issues. Planners can manually add issues to Plan elements, describe these issues and suggest remediations. Manually added issues can later be removed.

Issue detection and remediation can guide planners as to what to do next when building, fixing and improving a Plan. For example, a newly created segment with its lone task will immediately cause the detection of incompleteness issues. Resolving these incompleteness issues drives the planning process forward.

Some detected issues can be waived by planners if they are considered not to apply in a particular case. Note that when waiving an issue of a certain kind about a Plan element, all issues of this kind are also waived for that same element.

A planner can examine all issues detected or reported in the Plan and sort them by kind, by severity, by what they are about etc. to decide which ones to tackle next.
