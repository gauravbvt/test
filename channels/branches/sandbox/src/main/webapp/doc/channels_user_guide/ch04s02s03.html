  • Ease the time constraints the information need

  • or reduce the time delays of flows disseminating needed elements

Single agent assigned to critical task is bottleneck

Depends on impacted tasks

  • or add participating agents to the scope who will be assigned to the task

Yes

Agents assigned to task receive information from too many sources

Depends on impacted tasks

  • Remove incoming sharing flows

  • or add intermediates to spread the incoming communication load

Yes

Agents assigned to task have too many sharing commitments

Depends on impacted tasks

  • Remove outgoing sharing flows

  • or add intermediates to spread the outgoing communication load

Yes

Important task has only one assignment

Depends on impacted tasks

  • Profile agents so that more than one match the task specifications

  • or modify the task specifications so that it matches more than one agent

Yes

Availability gap in task assignments

Depends on impacted tasks

  • Change the availabilities of assigned agents so that they leave no gap

  • or change the specification of the task so that more agents are assigned to fill the gaps

Yes
