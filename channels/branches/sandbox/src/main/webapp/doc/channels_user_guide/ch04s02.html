ISP issues

Issues relate either to the validity, completeness or robustness of an IS plan. Issues are either detected by Channels or reported by planners (ad hoc issues).

Validity issues

The value given to an attribute of a Plan or one its elements is either inherently wrong or it causes a logical inconsistency.

Cause

Severity

Remediation

Can be waived

An agent is directly or indirectly his or her own supervisor

Low

  • Change the agent’s supervisor

  • or remove the agent from the supervisor hierarchy

Yes

Agent supervised by someone from another organization

Low

  • Transfer the supervisor to the agent’s organization or one of its parent (if any)

  • or transfer the agent’s job to the supervisor's organization or child organization (if any)

Yes

The source task of an information flow is undefined

Medium

  • Name the agent of the source task

  • or name the role

  • or name the organization

No

The target task of an information flow is undefined

Medium

  • Name the agent of the target task

  • or name the role

  • or name the organization

No

Can not put a place on a map

Medium

  • Change the geoname of the place

  • or remove it

No

Inconsistent impacts on a source task of sharing

Low

  • Specify that a flow of given information always terminates the task producing it

  • or specifies that it never does.

Yes

Inconsistent impacts on a target task of sharing

Low

  • Specify that a flow of given information always terminates or always triggers the task consuming it

  • or specifies that it never does.

Yes

Transmission medium delegates to inappropriate medium

Medium

  • Make the security of the delegated-to medium at least that of the delegated-from medium

  • and make sure that the delegated-to medium does not “widen” transmission (for example, from unicast to broadcast)

  • or do not delegate to the other medium.

No

Invalid channel

Low

  • If a duplicate, remove it

  • or make the end point valid

  • or change the end point pattern if the medium is not built-in.

No

Conflicting entity categorization

Medium

  • Remove conflicting type from categorization

  • or change definition of conflicting type to remove conflict

  • or change definition of categorized entity to remove conflict.

No

Medium transmission mode is unknown

Medium

  • Set medium transmission mode to either unicast (default), multicast or broadcast

  • or categorize the medium with another that has a set transmission mode.

No

Task's agent or role is named but not the organization

Medium

Name the organization of agents assigned to the task

Yes

Task repeats too soon

Low

  • Make the task complete sooner

  • or wait longer before it repeats

  • or make the task non-repeating

  • or set a completion time for the task.

No

Task without named role

Low

Name a role for the task.

Yes

Place contained in itself

Medium

  • Change the place this one is within

  • or define the place to be within none.

No

Postal code different from containing place

Low

  • Change the postal code

  • or remove it

No

Street address different from containing place

Low

  • Change the address

  • or remove it

No

Plan locale is relative

High

If the plan locale is a place that is within a place defined relative to another:

  • Change the definition of the containing place so that it is not relative

or if the plan locale is categorized with a place defined relative to another:

  • Change the categorization of the place type so that it is not relative

or, in either case,

  • use another place as the plan's locale

  • or leave the plan's locale unnamed.

No

Segment has no stated goal

Medium

  • Add a goal to the segment

  • or remove the segment

No

Segment has same goal as another segment

Medium

  • Merge the two segments

  • or move all tasks that achieve the common goal from one segment to the other and remove the goal.

No

Unnamed task started or terminated

Low

Name the task

No

Information flow is not named

High

Name the information flow.

No

The flow restriction is meaningless

Low

  • Remove the restriction

  • or change the source and/or target specification to use types of organizations or locations, or none at all

No

Sharing contradicts capability

Depends on impacted tasks

  • Modify the definition of the contradicted capability

  • or modify the definition of this sharing flow

No

Channel is redundant (e.g. Cell is redundant in a list containing Phone)

Low

  • Remove the redundant channel

  • or remove the channel that makes it redundant

No

Element of information is repeated

Low

  • Remove the repeated element of information

No

Contradictory policies

Medium

  • Remove all prohibiting policies

  • or remove all mandating policies

  • or remove all mandating policies and all prohibiting policies

No

Agent assigned to user with different name

Medium

  • Assign a user of the same name to the agent

  • or assign no user to the agent

  • or make the agent a place holder

  • or make the agent an archetype

Yes

Too many participations as agent

Medium

  • Unassign users

  • or make the agent an archetype

  • or make the agent a placeholder of which there can be more than one

No

More than one send can terminate the task

Medium

  • Have the task terminated by at most one send

Yes
