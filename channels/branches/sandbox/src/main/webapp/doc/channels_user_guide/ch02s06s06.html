Attaching documents

Plan elements are documented and annotated by attaching files to them. These files can either be uploaded to the Channels server, or they can be linked to from Channels if they are reachable over the Internet or over an intranet in which Channels operates.

Figure 2.47. Managing attachments


Documents can be attached, copied, or detached. Attachments are assigned a type. Certain types of attachment, for example, the picture type and the tags type impact the behavior of Channels (icons are generated and used, tags are added to the tag domain, etc.) Attachments can be named. The default name of an attachment is its URL.

Figure 2.48. Attaching and detaching documents


Uploading an attachment takes the planner to the platform's file chooser. Linking to a document requires only entering a URL (the http:// prefix is implied if not given).
