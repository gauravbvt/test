Abstract

This document introduces the key concepts of Information Sharing Planning, provides a reference guide to the Channels Web application, and shows how to apply the ISP Method with Channels to systematically build and analyze information sharing plans. A lexicon of ISP terms and the complete list of detectable ISP issues can be found in the Appendix.
