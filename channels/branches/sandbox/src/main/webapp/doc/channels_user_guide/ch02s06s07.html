Floating panels

Many panels in Channels are of the "floating" sort. They can be moved, resized, brought to the front and closed.

Figure 2.50. Floating panels


The following are floating panels:

  • Plan definition panel

  • Segment definition panel

  • Entity profile panel

  • Task assignments panel

  • Sharing commitments panel

  • Task overrides panel

  • Elements of information panel

  • Failure impact analysis panel

  • Dissemination analysis panel

  • Procedures map panel

  • Flow map legend panel

There can only be one entity profile panel opened at a time, only one task assignments panel, etc.
