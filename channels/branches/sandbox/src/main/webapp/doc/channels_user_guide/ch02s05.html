Administration page

A Channels administrator (and only an administrator) can create new plans, retire existing plans, move the development version of a plan to production (and thus retire the production version if any), and register, disable, and assign per-plan access privileges to users. Administrators are also responsible for configuring the Channels server.

Plan creation and user privileges

The Administration page is accessible only to administrators of the Channels server.
