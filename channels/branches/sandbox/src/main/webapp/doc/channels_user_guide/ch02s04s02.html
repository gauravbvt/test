Feedback panel

A feedback mechanism is available from the home page to all users of Channels. It allows users to ask questions, make recommendations and report problems. The feedback is sent via email to an address configured for the plan by an administrator. The email address of the user sending the feedback is included in the email. Further conversations can take place over email.
