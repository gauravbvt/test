Channels pages

Channels is a Web application is composed of a number of Web pages, some of which behave not unlike desktop applications.

The Channels pages are:

  • The login page

  • The "renew password" page

  • The home page

  • The administration page

  • The information sharing model page

  • The assignments and commitments page

  • The issues summary page

  • The participants list page

  • The IS guidelines page

  • and various pages to handle login failures, system errors, session timeouts etc.
