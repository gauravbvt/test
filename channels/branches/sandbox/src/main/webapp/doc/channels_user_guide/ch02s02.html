Channels Pages

Channels is a Web application composed of a number of Web pages, some of which behave not unlike desktop applications.

The Channels pages are:

  • The login page

  • The "renew password" page

  • The Home page

  • The administration page

  • The information sharing model page

  • The assignments and commitments page

  • The issues summary page

  • The participants list page

  • The IS guidelines page

  • Various other pages which handle login failures, system errors, session timeouts, etc.
