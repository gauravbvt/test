Completeness issues

An element of the IS plan is clearly missing given the scope of the plan and the elements it contains.

Cause

Severity

Remediation

Can be waived

Agent not member of any known organization

Low

Make the agent the member of an organization.

No

Sharing agreement encompassed by another

Low

Remove either sharing agreement.

Yes

Sharing agreement does not match any sharing commitment

Low

  • Remove the sharing agreement

  • or add flows that imply a commitment covered by the agreement.

Yes

Immediate or permanent task is unnecessarily triggered

Low

  • Have no flow trigger this task

  • or have the task not start with the plan segment.

Yes

Organization expected to have task assignments but has none

Low

  • Remove the expectation on the organization

  • or add a task that will be assigned to the organization or one of its sub-organizations, if any

No

Organization is missing expected commitment

Low

Fulfill expectation by having a task of a given category and assigned to an organization connected to another task of an expected category and assigned to an organization with the appropriate relationship, with a flow with an expected intent. For example, an audit task expects to receive a report from an operations task assigned to a child organization.

Yes

Organization is not assigned a category of tasks

Low

Specify at least one task of the missing category that is assigned to the organization or one of its sub-organizations.

Yes

Information capability or need has no element of information

Low

Add elements of information.

Yes

Sharing flow has no elements of information

Medium

  • Specify elements of information

Yes

Sharing commitment by an organization without required agreement

Low

  • Confirm an agreement covering this sharing commitment

  • or remove the requirement for agreements for the organization.

Yes

Sharing commitment over unicast channel with unknown endpoint

Depends on impacted tasks

  • Make sure that the agent has a known endpoint (address, phone number etc.) for contact via the unicast channel

  • or remove the unicast channel from the flow.

No

Agent shares information with self

  • Change the agent sharing the information

  • or change the agent receiving the information

Yes

Add a channel to the specification.

  • or remove phase from plan scope

  • Define a plan segment that responds to it

  • or break up the sharing flow.

Yes

Redundant task

Low

Yes

Redundant place

Low

  • Remove references to this place

  • or change its address

  • or change its geolocation.

Yes

Planned-for event may never be caused

Medium

  • Make the event an incident

  • or make sure at least one task in any segment causes it.

No

Segment’s event phase may never end

Medium

If the segment is for a co-event phase,

  • have the event end on its own

or if the plan segment is for a pre-event phase

  • have the event be an incident

  • or have a task in any segment cause it

or, in all cases,

  • have at least one task in the segment terminate the event phase.

Yes

Segment's event phase may never start

Medium

If the segment is for a co-event phase,

  • have the event be an incident

  • or have a task in another segment cause it

or if the plan segment is for a post-event phase

  • have the event be self-terminating

  • or have a task in a co-event segment terminate its segment.

Yes

Sharing flow implies no commitments

Medium

  • Modify the specifications for source and target tasks so that they both imply assignments

  • or add or redefine participating agents to match source and target task specifications

or, if inadequate clearances are at fault,

  • declassify elements of information

  • or make sure otherwise committed agents have the required clearances.

Yes

Task is triggered but never started

Depends on impacted tasks

  • If the task starts with its segment (it is permanent or immediate), Make sure the event phase of the segment can start (it is a co-event phase of an incident or the phase is started by a task that itself can start)

  • else, make that the triggering task itself can start.

No

Unconfirmed job of agent in organization

Low

Confirm the job in the organization’s profile

No

Ignored information need or capability

Low to medium

No

Unassigned task

  • or modify the task specifications so that it matches at least one participating agent.

Yes

Information need not fully satisfied
