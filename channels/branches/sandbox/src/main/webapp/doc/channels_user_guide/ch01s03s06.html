Secrecy, security and clearances

Planners can define one or more classification systems in an IS plan and use classification levels to annotate elements of information, transmission media and to set the security clearances of (actual) agents.

A secrecy classification has a name (e.g. “NATO”) and an ordered list of secrecy levels (“Restricted”, “Confidential”, “Secret”, “Focal Top Secret”, “Cosmic Top Secret”).

Figure 1.26. Secrecy classifications


Each classification system has a unique name and consists of a list of classification levels ranked from lowest (bottom) to highest (top).

Agents can be given security clearances. Clearances are expressed as classification levels (no more than one per classification system).

Figure 1.27. An agent with security clearance


How secure a transmission medium type is defined by the highest classification levels of the information it can transmit without infringing security requirements.

Figure 1.28. A secure transmission medium


Elements of information shared or to be shared can each be given classification levels as well as indications of the source and special handling instructions.

Figure 1.29. Classified elements of information


Secrecy and security issues that are automatically detected include:

  • Classified information transmitted over an insufficiently secure channel

  • A commitment to share classified information with an agent who has insufficient clearances

  • Receiving classified information and then disseminating it with reduced classification
