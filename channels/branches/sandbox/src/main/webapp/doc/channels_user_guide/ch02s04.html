Home Page

The Home page lets the user select a plan from all plans they are authorized to see. A plan is always selected by default upon logging in. The home pages of administrators, planners, and participants have different content reflecting their differing privileges in regards to the selected plan.

The administrator's home page is the same as a planner's home page except that the administrator is given access to the administration page.

Figure 2.3. Channels home page for administrators


A planner can

  • edit the development version of the plan and view its production version (same interface but edit form fields are disabled)

  • visually explore the task assignments and information sharing commitments (the IS procedures) generated by the plan

  • view metrics about issues detected in the plan

  • assign participants to agents in the plan and view their IS guidelines (IS procedures in report format)

  • View the information needs reports of all participants

Figure 2.4. Channels home page for planners


A participant in a normal (non-template) plan can only access the IS guidelines generated from the production version for the agent he/she is assigned to. A participant can not send messages to other users from within the collaboration panel, but is able to read messages broadcast to all users (or news) by planners (see below).

Figure 2.5. Channels home page for participants


An IS plan can be marked as a template. This indicates that there is nothing sensitive in the plan and that a participant can see not only his/her participation in the plan (personal IS guidelines, info needs report), but also the entire plan itself in its production version. A participant (and all other users) can see but not modify the production version of the plan.

Figure 2.6. Channels home page for participants of a "template" plan


Home page collaboration and settings panel

The collaboration and settings tabs in the home page allow planners to communicate with other users via messages, calendar events, and surveys.

The "About Me" tab lets a user edit his/her full name and private email, and change their password. If the user is a participant in the plan, the user can update the contact information for the agent he/she has been assigned to.

Participants can only read the broadcast messages from planners which appear in the "News" tab, but cannot reply to or comment on these messages.

Figure 2.7. News tab (participants only)


Planners (and thus administrators) can both send and receive messages.

Figure 2.8. Messages tab (planners and administrators)


Planners can add events to a Google calendar associated to the plan (as configured by administrators in the Administration Page). Events can be viewed by all users of the plan.

Figure 2.9. Calendar events for the plan


Planners can launch surveys and send them to users. A user can see all surveys received and the status of each survey, and can access a survey in order to respond to it.

Figure 2.10. Surveys sent to the user


A user can modify data related to his/her identity as a user but also, if applicable, data related to the agent the user is represented by in the plan.

Figure 2.11. User settings
